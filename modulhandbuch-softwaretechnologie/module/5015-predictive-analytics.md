## Kurzbeschreibung
**Deutsch:** Konzepte des Data Mining und Machine Learning für die Analyse betrieblicher Daten

**English:** Applying Data Mining and Machine Learning concepts

## Inhalte
- Mathematischer Grundlagen
- Bayessche Statistik
- Machine Learning Verfahren
- Entscheidungsbäume
- Regressionsmodelle (Linear, Logistisch)
- Clustering
- Grundlagen neuronaler Netze
- Weiterführende Modelle (z.B. Support Vector Machines)
- Typische Anwendungen (z.B. Warenkorbanalyse)
- Anwendung eines Standardwerkzeugs (BIRT Analytics,
- Entwickeln von Modellen mit R und Python

## Qualifikationsziele und angestrebte Lernergebnisse
### Fachkompetenzen
Die Studierenden lernen die Konzepte des Machine Learning und des Data Mining. Sie können bestehende Verfahren anwenden und auf spezifische Situationen übertragen. Die Studierenden können damit gegebene Fragestellungen aus der Praxis in ein Modell übertragen und die Ergebnisse interpretieren.

### Methodenkompetenzen
Die Studierenden sind in der Lage, Modelle mit Hilfe von Standardwerkzeugen oder mit Hilfe einer geeigneten Programmiersprache zu implementieren. Sie entwickeln Analysen mit Hilfe gängiger Werkzeuge (etwa R oder Python).
