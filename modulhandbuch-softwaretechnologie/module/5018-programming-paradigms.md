**English:** Fundamental concepts of exemplary programming languages

## Inhalte
- Principles of programming languages
- Abstraction of variables and control structure
- Memory Management
- Object Oriented paradigm
- Functional paradigm
- Logic programming paradigm
- Applications and typical problems

## Qualifikationsziele und angestrebte Lernergebnisse
The module consists of a mixture of lectures and tutorials. The lectures include discussion and demonstration.

### Fachkompetenzen
The students will solve problems with languages that follow different programming paradigms. They will thus experience how to choose and apply the appropriate programming language in a problem setting. They see advantages and disadvantages of the paradigms discussed in the lectures. Practical application of languages representing the paradigms discussed in the lectures (e.g. Scala, Python, Prolog, Lisp,…)

### Methodenkompetenzen
The students learn how to choose the most suitable programming paradigm for a given set of problems.

### Selbstkompetenzen
The students are able to follow complex topics in English.
