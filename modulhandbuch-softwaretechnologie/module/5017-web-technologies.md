## Kurzbeschreibung
**Deutsch:** Entwicklung von Web Applikationen

**English:** Programming Web based Applications

## Inhalte
- Web languages: HTML5, CSS3, JavaScript
- Essential protocols: HTTP and HTTPS, cgi, encoding
- Using jQuery to create content and respond to user activity
- Using jQuery to perform AJAX calls
- Programming with PHP and MySQL, generating web pages, generating JSON
- Security issues: authentication, encryption, hashing, salting
- Security exploits: SQL injection, cross site scripting, session hijacking
- Working with open-source resources for example: Bootstrap, jQuery-UI, WordPress, d3
- User interface issues: Standards compliance, HTML5 validation, WAI
- Hosting issues: scalability, monitoring performance, elastic capacity, load testing

## Qualifikationsziele und angestrebte Lernergebnisse
The module consists of a mixture of lectures and tutorials. The lectures include discussion and demonstration.

### Fachkompetenzen
The tutorials require students to build working applications. Students require access to a Linux platform to support Apache, MySQL, PHP.

Learning Goals and Outcomes — On completion students will be able to:

- Build web applications using client-side and server side components using JavaScript and PHP
- Apply event driven programming to produce effective and attractive user interfaces
- Identify and conform to web standards for validity and accessibility
- Develop secure applications that are not vulnerable to common exploits
- Integrate with open source libraries and frameworks

### Methodenkompetenzen
The students are able to follow complex topics in English.
