## Inhalte
- Grafische Benutzeroberflächen
- Oberflächenkomponenten
- Ereignismodell
- Layout
- Entwurfsmuster für die Programmierung
- Grafiken und Bilder
- Multithreading
- Animation
- Globalisierung
- Softwareergonomie:
- Mensch-Computer-Dialog (Psychologisches Grundwissen)
- Ergonomische Richtlinien und Normen
- Entwurfsmuster für den Arbeitsplatz
- Entwurfsmuster zur Werkzeugkonstruktion und –integration
- Softwareergonomie im WWW

## Qualifikationsziele und angestrebte Lernergebnisse
### Fachkompetenzen
Die Studierenden können Programm erstellen, die über grafische Benutzerschnittstellen bedient werden. Dabei können sie einen gegebenen Entwurf umsetzen.

### Methodenkompetenzen
Die Studierenden kennen
- grundlegende Strukturen und Mechanismen von grafischen Benutzeroberflächen,
- Entwurfsmuster für die Programmierung grafischer Benutzeroberflächen,
- die Problematik von nebenläufiger Aufgabenbearbeitung und Lösungsmöglichkeiten,
- psychologische Grundlagen für den Entwurf von ergonomischen Benutzerschnittstellen,
- ergonomische Richtlinien und Normen.

### Sozialkompetenzen
Die Studierenden können einen Entwurf im Dialog mit einem Anwender entwickeln.
